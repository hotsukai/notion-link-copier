# Privacy Policy / プライバシーポリシー

Last updated: 2026-04-01

## English

### Data Collection
Notion Link Copier does **not** collect, store, or transmit any personal data or browsing information.

### How It Works
- The extension reads the page title and URL of the currently open Notion page **only** when you click the copy button.
- The copied data is placed in your local clipboard and is never sent to any external server.

### Permissions
- **activeTab**: Used to access the current Notion page's title when you click the copy button. No background access occurs.
- **Host permission (notion.so)**: The extension only runs on Notion pages. It does not access any other websites.

### Third-Party Services
This extension does not use any third-party analytics, tracking, or advertising services.

### Changes to This Policy
If this policy is updated, the changes will be posted in this file in the GitHub repository.

### Contact
If you have questions about this privacy policy, please open an issue at:
https://github.com/hotsukai/notion-link-copier/issues

## 日本語

### データの収集について
Notion Link Copierは、個人データや閲覧情報の収集・保存・送信を**一切行いません**。

### 動作の仕組み
- コピーボタンをクリックしたときに**のみ**、現在開いているNotionページのタイトルとURLを読み取ります。
- コピーされたデータはローカルのクリップボードに保存され、外部サーバーには送信されません。

### 権限について
- **activeTab**: コピーボタンをクリックした際に、NotionページのタイトルとURLを取得するために使用します。バックグラウンドでのアクセスは行いません。
- **ホスト権限 (notion.so)**: 拡張機能はNotionのページでのみ動作します。他のウェブサイトにはアクセスしません。

### サードパーティサービス
この拡張機能は、アナリティクス・トラッキング・広告などのサードパーティサービスを一切使用しません。

### ポリシーの変更
本ポリシーが更新された場合、GitHubリポジトリ内のこのファイルに変更が反映されます。
